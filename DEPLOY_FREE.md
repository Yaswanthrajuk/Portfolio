# Free Deployment Guide

Your portfolio is a static website, so it can be hosted for free.

## Fastest: Netlify Drop

1. Open https://app.netlify.com/drop
2. Drag this full folder into the page:
   `yaswanth-portfolio`
3. Netlify will give you a public link.

## Professional: GitHub Pages

1. Create a GitHub repository named `yaswanth-portfolio`.
2. Upload everything inside this folder.
3. Go to `Settings > Pages`.
4. Select your main branch as the source.
5. Your link will look like:
   `https://YOUR_USERNAME.github.io/yaswanth-portfolio/`
